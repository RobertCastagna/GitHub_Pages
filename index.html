Quant Trading App

I entered into an options trading competition on the TSX with a colleague. His skillset revolved around portfolio management whereas I brought to the team the expertise in mathematical modelling and software engineering. With his direction on general strategy, I was able to put together a web app for mobile and desktop leveraging Streamlit's framework. It has the ability to pull 15-min delayed data from Yahoo Finance's API and based of a series of backtested strategies, signal good entry points for an investment guide. There is also the capability to use Black Scholes and Binomial options pricing models on a second page. Features were continually added over the course of the 8 week competition. You are able to find the full Investment Policy Statement on my LinkedIn and the project description on my Github linked at the top of this page.

Data Orchestration Project

This is a data pipeline orchestrated by Dagster leveraging two API's, Snowflake storage and a linear regression model. The process reads the top 20 holdings of the S&P 500 index from one API, then sends that data to IEX Cloud's API to retrieve the price history. All historical data lives in Snowflake and is read on command into a lin. regression model to predict future movement in price. The results of this model output are moved to S3, where they can be built into various other applications. The motivation for this project was to learn how to leverage a workflow orchestration tool to have a level of abstracted control that stemmed across applications in a large corporate setting. This problem arose in my job and I decided to experiment with potential technologies as a solution and built out a proof of concept.
